# Stable v3.1.5 — Changelog

Release date: 2026-04-13

---

## Bug fixes

### Dropdown lists not scrolling with mouse wheel / trackpad
Added `onWheel` event propagation stop to all popover result lists across Orders, Quotations, Tasks, and Leads. Previously, scrolling with a trackpad or mouse wheel over a dropdown would scroll the page behind it instead of the list itself.

### PWA "Check for Updates" always showing "Up to date"
`reg.update()` resolves as soon as the service worker file is fetched — but the new worker needs additional time to reach the `installed` state. The check was reading `__swUpdateAvailable` too early and always seeing false. Fixed by waiting up to 4 seconds after `reg.update()` for the `swUpdateAvailable` event before concluding no update is available.

---

## Changes

### Update button UI redesign
The version / update section in Settings is now a full-width stacked layout. The "update available" state is shown as a pill badge next to the version number. The action button is full-width to prevent text overflow on narrow screens.
