# v3.2.6 — Changelog

Release date: 2026-04-28

---

## Changes

### Quotations: Removed Notes field from form
The Notes (internal notes) textarea has been removed from the quotation create/edit form. Terms & Conditions remains unchanged.

### Quotations PDF: Rupee symbol on all amount values
Added ₹ prefix to every amount in the PDF — individual item amounts, subtotal, discount, and total rows.

### Quotations PDF: Amount column widened
Increased the Amount column width from 66pt to 82pt to prevent values from feeling cramped.

### Quotations PDF: Footer no longer jumps to next page prematurely
Moved `wrap={false}` from the entire footer to only the right-side totals+signature column. The left side (bank details, terms) now flows naturally, so the footer no longer gets pushed to a new page when there is sufficient space remaining.
