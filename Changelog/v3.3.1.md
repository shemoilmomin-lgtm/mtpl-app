# v3.3.1 — Changelog

Release date: 2026-05-04

---

## Changes

### Mobile: Logo Size Increased
The MTPL logo in the mobile top bar has been increased in size and top padding adjusted for better visual balance.

### Orders: Proforma, Invoice & Delivery Date Editable by All Roles
The pencil edit icons for Proforma Invoice Number, Invoice Number, and Delivery Date in the order view were incorrectly restricted to admin and superadmin roles only. Executives can now edit these fields as well.
