## Bug fixes

### Quotation combobox showing legacy timestamp IDs
All 33 quotation records created before the MTPLQ naming convention was adopted had `q_<timestamp>` or `QT-xxxx` format IDs. A database migration converts all non-MTPLQ quotation IDs to `MTPLQ-xxxx` format using the record's id (e.g. id=5 → MTPLQ-0005). The quotation combobox in the order edit form now shows clean MTPLQ numbers for all records.
