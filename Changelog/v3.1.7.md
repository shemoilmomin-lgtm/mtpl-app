# Stable v3.1.7 — Changelog

Release date: 2026-04-13

---

## New features

### Due date for tasks
Tasks now support a due date field. The `due_date` column already existed in the database schema but was not wired to the UI or API.

- **Create / edit form**: A "Due Date" date input is shown under the Status field.
- **Task view panel**: Due date is displayed in the Details tab. Overdue dates (past today, task not done) are highlighted in red.
- **Desktop table**: The "Date" column now shows the due date instead of the created date. Overdue due dates are highlighted in red.
- **Mobile card**: Due date is shown as a small label beneath the assignee / client info. Overdue dates are highlighted in red.
- **Status change**: `due_date` is now preserved when changing a task's status inline (it was previously being cleared by the status-change save call).
