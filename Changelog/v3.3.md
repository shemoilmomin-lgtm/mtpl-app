# v3.3 — Changelog

Release date: 2026-05-03

---

## Major Changes

### User Profile Photos
Users can now upload a profile photo from the Settings → Preferences tab. Photos are stored in Cloudflare R2 and are visible to all users across the app — in the sidebar, comments, recent comments panel, mention dropdowns, task assignees, and the live users list. Falls back to initials if no photo is set.

### Login Screen Redesigned
The login screen has been visually redesigned with an updated layout, improved typography, and a cleaner overall presentation.

### Sidebar and App Styling Revamped
The sidebar, top bar, and general app chrome have been restyled. The top bar height is now consistent across all pages, logo alignment is corrected on mobile, and spacing throughout has been tightened and refined.

### Comments Added to Leads Module
The Leads detail panel now includes a full comments section, allowing users to leave notes, reply to threads, and mention teammates — consistent with Orders and Tasks.

---

## Minor Changes

### Comments: Duplication Issue Resolved
Comments were appearing duplicated in certain scenarios. This has been fixed — each comment now appears exactly once.

### Comment Input: Auto-collapse After Submit
The comment textarea now resets to single-line height after a comment is submitted, instead of retaining the expanded height from the previous input.

### Comment Input: Outline Removed
The extra border/outline around the comment input field has been removed for a cleaner look.

### Recent Comments Panel: Font Sizes Increased
Text in the Recent Comments (activity feed) panel is slightly larger for improved readability on both mobile and desktop.

### Dashboard: Orders Chart Now Shows Rolling 13 Months
The Orders chart previously showed the fixed Jan–Dec window of the current year. It now shows the current month plus the 12 months prior, giving a more useful rolling view of order activity.

### Dashboard: Archived Tasks Excluded from Counts
Archived tasks are no longer included in the dashboard task counts and breakdown. Only active (non-archived, non-deleted) tasks are counted.

### Tasks: Archived Tasks No Longer Appear in Dashboard
The `/api/tasks` endpoint now filters out archived tasks, so the dashboard stat cards and task breakdown reflect only active work.
