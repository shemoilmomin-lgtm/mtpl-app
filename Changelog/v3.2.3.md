# v3.2.3 — Changelog

Release date: 2026-04-22

---

## Improvements

### UI: page size dropdown redesigned across all modules
Replaced the shadcn Select component used for "Show 20 / 50 / 100" pagination controls in Orders, Leads, Tasks, and Clients with a DropdownMenu matching the backdrop blur style used throughout the rest of the app. Fixes the two-line wrapping issue and removes the blue focus ring on the selected item.

### Comments: textarea scroll and styling fixes
The comment input textarea in Orders, Tasks, and Leads now has a maximum height with scrolling enabled when content exceeds it. Background changed to white (light mode) / dark (dark mode). Scrollbar is now slim and unobtrusive. Border and focus state are handled cleanly by the wrapper — single consistent border that subtly darkens on focus with no thickness change.
