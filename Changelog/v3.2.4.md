# v3.2.4 — Changelog

Release date: 2026-04-23

---

## Bug Fixes

### Orders: Job type dropdown now scrollable
The job type `SelectContent` was using an unbounded CSS variable for max-height, preventing scrolling. Added `max-h-60` so the dropdown shows a fixed-height scrollable list, consistent with the status selector behaviour.

### Comments: @mention user dropdown now appears correctly
The `MentionInput` wrapper had `overflow-hidden` set, which clipped the absolutely-positioned user dropdown popup before it could be seen. Removed `overflow-hidden` from the wrapper — the rounded border styling is unaffected. @mention tagging now works across all comment sections (Orders, Tasks, Leads, Clients, Quotations).
