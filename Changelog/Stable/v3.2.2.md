# v3.2.2 — Changelog

Release date: 2026-04-22

---

## New features

### Orders: "Work in Progress" status
Added a new order status — Work in Progress — with an indigo colour theme. Appears in all status dropdowns, filters, and badges.

### Orders: inline delivery date editing in view mode
The Delivery field in the order view panel now has an inline edit button (matching the Proforma and Invoice No. fields), allowing admins to update the delivery date without opening the full edit form.

### Quotations: Show All pagination option
Added a "Show All" option to the quotations page size dropdown, allowing all quotations to be displayed at once without pagination.

### Quotations: sort by ID
Quotations are now sorted by ID (newest first) instead of alphabetically by quotation number, ensuring correct ordering regardless of the prefix used (QT- or MTPLQ-).

---

## Bug fixes & improvements

### Orders: proforma and invoice edit pencil always visible
The pencil icon next to Proforma and Invoice No. in order view mode was previously only visible on hover. It now shows at all times.

### Quotations: quotation number sourced directly from database
Removed frontend fallback logic that derived quotation numbers from the record ID. All quotation numbers are now read directly from the `quotation_id` column, which has been backfilled with correct QT-XXXX values for all legacy records (ID ≤ 41).

### Quotations: page size dropdown redesigned
Replaced the shadcn Select with a DropdownMenu matching the blur/popover style used elsewhere in the app. Fixes the selected-item blue outline and the two-line wrapping issue where "Show" and "20" appeared on separate lines.

### UI: favicon updated
Browser tab and PWA icon updated to the MTPL favicon.

### UI: theme-aware logo
The sidebar and mobile header logo now correctly shows the colour logo in light mode and the white logo in dark mode, instead of using a CSS invert filter.
