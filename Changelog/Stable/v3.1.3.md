# Stable v3.1.3 — Changelog

Release date: 2026-04-13

---

## Bug fixes

### Proforma & Tax Invoice columns hidden on iPad landscape
The Proforma No. and Tax Invoice No. columns were hidden below 1280px (xl breakpoint). Removed the visibility gate — both columns are always shown, with horizontal scroll available if the table is too wide.

### Comment edit & delete buttons missing in Quotations
The Quotations comments tab was missing edit and delete buttons entirely. Now matches the behaviour of all other modules: own comments can be edited within 2 hours and deleted within 30 minutes. Admins can edit or delete any comment at any time. Edited comments show an "(edited)" label.

### Quotation search in Orders edit mode searching wrong field
The quotation combobox in the order edit form was only searching by the internal `quotation_id` reference number. It now searches across all relevant fields: quotation ID, subject, manual client name, linked client company/full name, and total amount.

### Quotation combobox showing opaque reference numbers
Each item in the quotation selector now shows the quotation ID, client name, and subject together, making it easy to identify the right quotation. The selected value label also includes the client name and total amount.
