# Stable v3.1.8 — Changelog

Release date: 2026-04-16

---

## New features

### Comment edit and delete in Leads and Clients
Leads and Clients now support editing and deleting comments, bringing them in line with Orders, Tasks, and Quotations.

- **Edit**: Authors can edit their own comments within 2 hours. Admins can edit any comment. An inline textarea replaces the comment text with Save / Cancel actions. Edited comments are marked with an "(edited)" label.
- **Delete**: Authors can delete their own comments within 30 minutes. Admins can delete any comment.

---

## Bug fixes

### Global search results now open the clicked record
Clicking a search result now correctly opens the specific record in its detail panel. Previously, if you were already on the target page (e.g. already viewing Orders), the navigation would land on the page but the record drawer would not open. The fix applies to Orders, Tasks, Clients, Quotations, and Leads.
